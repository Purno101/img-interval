The Ultimate Comparision.

--By Purno

Each of the slider bellow has an interval of 3 second and 5 second respectively. There are 36 images for each slider and the images are some for both. When a slider hits the last image, it will loop to the 1st one. The question is, if both of the slider starts at the same time with the same image, what will be the least amount of time needed for both of them to show same image again. It can be any image (not just the first one) meaning any two similar images.

Press The button to start the slider.
